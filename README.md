# Application Monitoring & Incident Management System

## Overview

This project simulates the responsibilities of an Application Support Engineer by monitoring application logs, validating database records, analyzing incidents, and identifying recurring issues using SQL.

## Objectives

- Monitor application health
- Validate application data
- Investigate incidents
- Perform root cause analysis
- Track support metrics

## Technologies

- Excel
- Git
